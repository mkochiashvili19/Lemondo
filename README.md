# ბუღალტერი - ფინანსური მართვის აპლიკაცია

## 1. არის თუ არა Back-end გაწერილი?
არა, ეს არის **Frontend-only** (მხოლოდ წინა მხარის) აპლიკაცია. მას არ აქვს რეალური სერვერი (მაგ: Node.js) და მონაცემთა ბაზა (მაგ: PostgreSQL). 

**თუმცა, დავამატე სიმულირებული ბექენდი (localStorage):**
იმისათვის, რომ აპლიკაცია რეალურად გამოგადგეთ და მონაცემები არ დაიკარგოს გვერდის დარეფრეშებისას, `DataContext.tsx` ფაილში დავამატე ბრაუზერის მეხსიერების (`localStorage`) გამოყენება. ახლა, როდესაც დაამატებთ ახალ ინვოისს ან ხარჯს, ის შეინახება თქვენს ბრაუზერში და საიტზე ხელახლა შესვლისას ისევ იქ დაგხვდებათ.

## 2. როგორ ავტვირთოთ Netlify Drop-ზე?
Netlify Drop-ზე პირდაპირ `.tsx` (TypeScript და React) ფაილების ატვირთვა არ იმუშავებს, რადგან ბრაუზერებს არ ესმით ეს კოდი. ჯერ საჭიროა კოდის დაკომპილირება (Build).

ამისათვის დაგიმატეთ საჭირო კონფიგურაციის ფაილები (`package.json`, `vite.config.ts`). მიჰყევით ამ ნაბიჯებს თქვენს კომპიუტერში:

1. დარწმუნდით, რომ გიყენიათ **Node.js** (გადმოწერეთ nodejs.org-დან).
2. გახსენით ტერმინალი (Terminal/CMD) ამ პროექტის ფოლდერში.
3. გაუშვით ბრძანება მოდულების ჩამოსატვირთად:
   ```bash
   npm install
   ```
4. გაუშვით ბრძანება პროექტის დასაკომპილირებლად:
   ```bash
   npm run build
   ```
5. ამის შემდეგ პროექტის ფოლდერში შეიქმნება ახალი ფოლდერი სახელად **`dist`**.
6. **სწორედ ეს `dist` ფოლდერი უნდა გადაიტანოთ (Drag & Drop) Netlify Drop-ის საიტზე.**
